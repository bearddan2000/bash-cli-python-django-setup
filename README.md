# bash-cli-python-django-setup

## Description
Creates a simple django project
with landing page and 2 apps.

## Tech stack
- bash

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
http://localhost:8000

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
